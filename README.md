**LMU RPC Mod** is a standalone application that brings **Discord Rich Presence** to **Le Mans Ultimate**. Show your friends exactly what you are driving, where, and how fast, with real-time updates and automatic detection.

##  Features

*   **Automatic Detection:** Instantly detects Car, Track, Class, and Session type.
*   **Smart Status:** Shows "In Menus", "Qualifying", "Race", or "Practice".
*   **Real-Time Telemetry:** Displays current Lap, Position, and Time Remaining.
*   **Multi-Language:** Fully translated into English, Portuguese, and Spanish (Auto-detected).
*   **Auto-Start:** Option to start automatically with Windows.
*   **Zero Config:** Just open and play. No manual IP or port configuration required.

## 📥 Installation

1.  Go to the [**Releases**](https://github.com/uWaazy/LMU-RPC-Mod/releases) page.
2.  Download the latest `.zip` file (e.g., `LMU_RPC_v1.0.zip`).
3.  Extract the file anywhere on your PC.
4.  Run `LMU_RPC_v1.0.exe`.
5.  Open **Le Mans Ultimate** and enjoy!
