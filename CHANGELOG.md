# Changelog

## v0.1 (Release) - 2026-01-23
- Initial release of LMU RPC Mod.
- Full integration with Discord Rich Presence.
- Automatic detection of vehicles, tracks, classes and sessions.
- Dynamic display of Safety Rating (Rank).
- Multi-language support (Portuguese, English, Spanish).
- Autostart functionality with Windows.
- Inclusion of the Manual Editor for personalized statuses.
